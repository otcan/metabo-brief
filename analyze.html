Run the curated SNP panel without uploading genotype data.

MetaboBrief reads 23andMe-style and AncestryDNA-style raw files in the browser, matches direct rsID/genotype calls against the open evidence panel, and renders a local report with source links and method limits.

01 Parse raw SNP file

Accepts TXT, CSV, TSV, ZIP, and gzip consumer exports.

02 Match direct genotypes

No strand flipping, imputation, phasing, or liftover.

03 Render evidence report

Review status, effect direction, caveats, and source links stay inspectable.
