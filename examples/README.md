# Examples

This directory is reserved for examples that demonstrate MetaboBrief structure without exposing personal data.

## Planned examples

- A synthetic metabolomics summary.
- A synthetic pathway-hypothesis report card.
- A matching machine-readable JSON payload for the report preview.

## Example rules

- Use synthetic data by default.
- If using public data, document source, license, consent, and redistribution terms.
- Do not include raw genotype files, raw metabolomics files, names, contact details, or private reports.
- Keep source attribution visible for any scientific claim.
