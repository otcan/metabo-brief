Improve the template

MetaboBrief accepts improvements through GitHub issues and pull requests. Keep examples synthetic and avoid personal health data.

Good contribution areas

  • Report sections that better explain uncertainty, limitations, and validation markers.
  • Accessible UI improvements for the report preview and evidence explainer.
  • Example content that uses synthetic metabolomics or genotype data.
  • Documentation for deploying the static site with GitHub Pages.
